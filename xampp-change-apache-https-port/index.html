XAMPP更改Apache HTTPS連線預設端口443 Port教學

在上一篇文章，硬兔有教各位如何更改XAMPP Apache HTTP連線的端口，既然Apache的HTTP連線端口都能夠更改，那HTTPS的連線端口是不是也能夠更改呢?當然是可以的呀!在這篇文章中，硬兔將要教你如何更改XAMPP Apache HTTPS的連接端口，像是指定成8443 Port。


首先，在XAMPP的安裝資料夾內找到並進入conf和extra資料夾，並將extra資料夾內的httpd-ssl.conf這個檔案開啟(硬兔依舊是建議使用NotePad++這個軟體開啟)。

 

之後在第36行找到Listen 443這行文字，並將Listen後面的數字更改成你想要使用的端口。例如硬兔想要把XAMPP Apache HTTPS的連線端口設定為8443 Port，則將該行文字更改成Listen 8443。

 

接著到第121行找到<VirtualHost _default_:443>這行文字，並將冒號後面的數字更改成你在上方所使用的端口。

 

然後看到下方的第125行，將ServerName example.com:443這行文字更改成你的網域名稱以及指定的端口。(由於HTTPS連線時會使用到SSL憑證，因此需要將example.com更改成你的SSL憑證所使用的域名(Domain Name)。)

 

都設定好之後，將檔案儲存。

 

再來開啟XAMPP Control Panel並按下Config。

 

接著進入Service and Port Settings。

 

然後將SSL Port更改成你在上方所指定的端口並按下Save。

 

接著再次按下Save。

 

將Apache重新啟動後，就可以看到XAMPP Apache HTTPS連線的端口已經更改成你所指定的Port囉!

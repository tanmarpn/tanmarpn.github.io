在CMD、BAT使用START執行任意檔案、開啟網站網址或程式

在CMD或BAT批次命令檔中要怎麼去開啟一個.EXE的執行檔，又或者是開啟一張圖片、一部影，甚至是開啟一個已經寫好的.txt說明文檔?當你在執行完命令檔之後想開啟一個你的網站，又或者顯示操作說明的文件給使用者看，你該怎麼做?硬兔在這裡將要教你一些常用的小技巧。


首先，開啟CMD命令提示字元。

 

再來輸入start 要開啟的檔案路徑，並按下Enter。例如硬兔要開啟位於C槽內名為in2site_app.exe的這個應用程式，則輸入 :

start c:\in2site_app.exe

 

那要如何開啟一張圖片呢?例如硬兔要開啟位於C槽內名為in2site_smile.jpg的這張圖片，則輸入 :

start c:\in2site_smile.jpg

 

在執行完上方這個指令之後，你可能會發現start指令所開啟的檔案，是以系統的預設程式去開啟指定檔案，像是上方這段指令有些人的照片可能是以windows內建的相片檢視器開啟，有些人可能是以自己另外安裝的圖片檢視軟體做開啟。那你一定會想問有沒有辦法以指定的程式開啟檔案呢?當然也是有辦法的阿!

 

我們如果想要以特定的程式去開啟指定的檔案時，我們只需要輸入start 程式路徑 要開啟的檔案路徑，就可以了。例如硬兔要以記事本開啟位於C槽內名為in2site_readme.txt的這個說明文件，則輸入 :

start notepad c:\in2site_readme.txt

 

至於要怎麼開啟一個網站或網址呢?例如硬兔要開啟https://in2.site/這個網址，則輸入 :

start https://in2.site/

Windows 10將我的電腦新增到桌面教學

如果你是第一次使用Windows 10肯定會發生一個問題，那就是找不到我的電腦這個捷徑，想要進去C槽或D槽或USB隨身碟內看資料，卻找不到要從哪裡進去，到底要怎麼才能我的電腦裡面阿?硬兔在這裡就要告訴你，Widnows 10裡面的我的電腦到底藏在哪裡，又該如何將我的電腦的捷徑新增到電腦桌面(桌布)上。


首先，在桌面上的任何一個地方點擊滑鼠右鍵，並按下個人化。

 

接著進入左方的佈景主題。

 

然後按下右邊的桌面圖示設定。

 

將電腦打勾，然後按下套用。

 

按下確定之後，我的電腦的捷徑就會新增到桌面囉!(Windows 10的我的電腦的捷徑名稱，已更改為本機)

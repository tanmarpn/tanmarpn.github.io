網站或網址被Facebook封鎖解決辦法

當你在Facebook上發布網站網址、張貼你的網頁連結時，如果出現了「你無法發布此內容，因為內含封鎖連結」，你該怎麼辦?你一定會想說我的網站又沒怎樣，怎麼會無緣無故被臉書封鎖?我該如何解鎖?又該到哪裡聯絡誰，把問題反映給臉書?硬兔在這裡就要大家如何去向Facebook申訴，如果順利的話，你的網站將會在三天內被解鎖，讓你在Facebook上能夠順利地張貼你的網頁連結。


在我們向Facebook提出申訴之前，必須要了解為甚麼臉書會封鎖你所要張貼的連結。

  1. 網頁內容不適當 : 可能你的網站、網頁內含有情色、非法內容，或者網頁內含不當程式碼。

  2. 網頁包含已遭到臉書封鎖之連結 : 在你的網頁內，你可能使用到一些已經遭到臉書官方所封鎖的連結，這將導致你的網頁也遭到封鎖。

  3. 使用他人的子域名(Sub-Domain) : 如果別人使用跟你相同的頂級域名所產生的子域名或者該頂級域名本身就遭到臉書封鎖，那你所使用的子域名有極大的可能一併遭到封鎖。

  4. 在臉書濫發你的網頁連結 : 如果你在短時間內，將你的網頁連結大量的張貼到臉書動態或社團或其它地方，是很有可能導致被臉書封鎖的!當然，如果是別人在短時間內大量張貼你的網站連結，也是會被當成垃圾網站封鎖的!

  5. 運氣不好 。

 

如果你有發生上述第1點與第4點的情形，請先改善，不然就算你申訴100次也不會有用的!如果沒有的話就跟著硬兔申訴去吧!

首先，開啟這個網頁。

https://www.facebook.com/help/contact/244560538958131

在當你無法分享連結時，請複製並貼上完整的網址的地方貼上網頁網址。

 

之後點選這是我擁有/推廣的網站。

Sony手機無法開機或一直重開機解決辦法

奇怪，為甚麼我的Sony手機不能開機甚至是一直重開機變成無限重啟?為甚麼開機到一半就卡住整個當機當掉了，完全無法進入主畫面?遇到這種情形時該怎麼辦?難道只能把手機送修、送回原廠修理嗎?


當我們遇到這種情形時，內心一定是非常著急、害怕的，無非是為了手機的維修費用以及儲存在手機中的資料而擔憂。現在你只需要靜下心來並且跟著硬兔一起做，如果夠幸運的話，你就可以在短短的幾秒內將手機無損的救回來囉!

 

首先硬兔要先教各位一些簡單的Sony手機故障排除方法 :

  1. 移除SIM卡 : 將你的手機SIM卡(信卡)取出，並且重新開機。

  2. 移除記憶卡 : 將手機內的記憶卡(SD記憶卡)取下，並且重新開機。

  3. 移除所有外置設備 : 將手機除了電池以外的裝置都移除，並且重新開機。

 

經過上面的故障排除辦法，相信許多人的手機已經可以順利開機了，但如果你不是那些幸運者中的其中一人，也不要太灰心，硬兔還有準備另一個辦法來幫助你救回你的愛機。

 

首先，使用電腦進入下方網址。

https://support.sonymobile.com/tw/xperia-companion/

再來依照你的作業系統下載Xperia Companion，如果你不知道你電腦的作業系統也沒關係，硬兔教你一個簡單的判斷方法，如果你的電腦並非蘋果(APPLE)的產品，那就是要下載Windows版本，反之則下載Mac OS版本。

 

下載好Xperia Companion並且安裝後，將它打開。

 

接著按下軟體修復。

 

選擇Xperia手機或平板。

 

到了這裡有幾點事情需要注意 :

  1. 手機在整個修復過程中是無法進行充電的，所以你必須讓手機擁有足夠的電量(至少80%)，避免修復過程中手機整個沒電造成不必要的麻煩。

  2. 如果你不確定手機是否有足夠的電量，請先將手機充電直到充電提示燈亮起綠燈。

  3. 如果你是使用筆記型電腦，也請為你的筆記型電腦接上充電線，避免操作過程過久而導致電量用盡。

 

如果你準備好了的話，就按下下一步吧。

 

接著勾選是，我知道我的Google帳號詳細資料，並按下下一步。

 

再來等它初始化一下。

 

接下來的這個步驟是最關鍵也是最多人向硬兔求救的地方!請先詳閱操作步驟後，再行操作!

  1. 首先必須將你的手機關機，若手機可以自行拔除電池，建議直接將電池拔除，並等待5秒後再將電池裝回。若手機無法順利關機，請參考硬兔所寫的這篇文章「Sony Xperia手機當機沒反應，強制關機教學」。

  2. 將USB線(充電線)插入電腦(先不要將線插入手機)。

  3. 按住手機的音量降低鍵。

  4. 將USB線插入手機。

注意 : 第3點的「按住手機音量降低鍵」指的是持續按壓，而不是按一下而已。請持續按壓音量降低鍵，直到畫面進入到下一步驟。如果你的畫面依舊是跟上方這張圖片一樣，請重新照著步驟執行，如果多次皆無法成功，建議換一條USB線(不要使用行動電源所附贈的)或是將USB線插在電腦的不同USB插槽。

 

之後只要讓它跑一段時間，手機就能完全修復好囉!

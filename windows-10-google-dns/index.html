Windows 10使用Google DNS幫網路、網頁加速

你是否感覺瀏覽網頁的速度很慢，每次開個網站或是網頁都要等很久才會跑完?硬兔今天要來跟大家講一個好康的小撇步，只要將電腦、Windows 10的DNS伺服器更改成Google Public DNS免費DNS伺服器，就可以幫你的網路加速囉!讓你在瀏覽網頁的時候可以更順暢、獲得更低的延遲，讓網頁加載速度變快。


首先，開啟開始功能表，並按下左下角的齒輪。

 

再來按下網路和網際網路。

 

然後按下變更介面卡選項。

 

在你目前使用的網路上點及滑鼠右鍵，並按下內容。(如果有多個網路選項並且你不知道是目前是使用哪個網路，可以照圖片設定，或是每個網路都設定一次。)

 

點選網際網路通訊協定第 4 版(TCP/IPv4)，然後按下內容。

 

然後點選使用下列的 DNS 伺服器位址。

 

在慣用的 DNS 伺服器處輸入8.8.8.8，並在其他 DNS 伺服器輸入8.8.4.4，然後按下確定。

 

按下關閉後就可以幫你的網路加速，讓你瀏覽網頁的速度提升囉!

在CMD、BAT使用批次命令檔使用COPY複製檔案

在CMD命令提示字元或BAT批次命令檔中要怎麼複製指定的檔案?要怎麼使用或是透過CMD指令複製檔案到其它地方?在BAT批次命令檔中需要將特定的檔案複製到別的資料夾中，要怎麼做?硬兔在這篇文章要教你，在CMD或BAT的環境下如何複製檔案到其它資料夾或任何路徑、地方。


首先，開啟CMD命令提示字元。

 

再來輸入copy 要複製的檔案 複製檔案目的地，並按下Enter。例如硬兔要將C槽內名為in2site.txt這個檔案，複製到D槽並取名為in2site_2.txt則輸入 :

copy C:\in2site.txt D:\in2site_2.txt

 

又或者我們想要將一個資料夾內的所有JPG圖片複製到其它地方，則可以借助萬用字元來實現。例如硬兔想要將C槽內image資料夾中所有JPG圖片，複製到D槽內的backup資料夾，則輸入 :

copy C:\image\*.jpg D:\backup

 

那你一定會問說，欸硬兔竟然可以一次將資料夾內的所有圖片複製到其它資料夾中，那有沒有辦法將一個路徑(資料夾)下的所有檔案複製到指定的資料夾中，來實現備份檔案的功能?當然是有啊，例如硬兔要將C槽內mix資料夾中的圖片、照片、文件等所有檔案，備份(複製)到D槽backup資料夾裡面，則輸入 :

copy C:\mix\*.* D:\backup

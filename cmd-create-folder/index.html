在CMD、BAT批次檔使用MD新增、建立資料夾

在上一篇文章中，硬兔有教你如何使用RD指令刪除指定的資料夾，你一定很好奇，竟然有刪除資料夾的指令，那有沒有新增資料夾(目錄)的指令呢?答案是當然有阿!而且這個指令通常會在我們複製檔案的時候被用到，至於要如何操作呢?就跟著硬兔繼續看下去吧!


首先，開啟CMD命令提示字元。

 

再來輸入md 資料夾名稱，並按下Enter。例如硬兔要在C槽內建立一個名為in2site的資料夾，則輸入 :

md c:\in2site

 

那如果你是要一次新增多個資料夾呢?你一定會問硬兔難道我必須為每一個資料夾分別輸入一次md指令嗎?當然是不用阿!我們只需要一行的md指令，並且在要建立的每個資料夾間加入一個空白鍵，這樣就可以囉!例如硬兔要在C槽內建立名為in2site_1與in2site_2的這兩個資料夾，則輸入 :

md c:\in2site_1 c:\in2site_2

 

到了這裡你可能又會想問硬兔，那如果我要新增一個資料夾，並且在該資料夾內再新增一個子資料夾，要怎麼做會比較好呢?我需要為兩個資料夾分別輸入md指令嗎?當然也是不用呀!我們只需要建立最終的子目錄就可以了，其它的上層(中間)資料夾系統會幫你自動建立。例如硬兔要在C槽建立一個名為in2site_parent的資料夾，並且在in2site_parent資料夾內新增一個名為in2site_child的資料夾，則輸入 :

md c:\in2site_parent\in2site_child
